REV 2026-05 · v1.0 █

법무법인 도원 웹사이트에 구현된 AI 에이전트 8종 + 서브루트 1종. 각 에이전트는 Claude (claude-opus-4-7 계열) 위에서 동작하며, NDJSON 스트리밍 · 레이트 리밋 · 감사 로그를 공통 인프라로 사용합니다.

▌ AGENTS · 8 EA

AI #01 · /api/ai/triage
Triage
사건 유형 진단 챗봇
public

의뢰인이 자유롭게 상황을 서술하면 사건 카테고리(교통·의료·건설·상속·기타)를 분류하고, 후보 변호사·관련 라이브러리 항목을 함께 안내합니다. Intake로 인계하는 입구 역할.

IN 사용자 메시지 + 대화 이력 → OUT 분류 + 변호사 후보 + 다음 단계 추천
CHATSTREAM RL · lightAUDIT
AI #02 · /api/ai/intake
Intake
사건 정보 수집 도우미
public

사건 정보를 슬롯 기반으로 채워가는 멀티턴 챗. 일시·장소·당사자·손해·자료 등을 단계적으로 묻고, 완성도(completeness)가 50% 이상이면 상담 신청으로 넘길 수 있습니다. localStorage로 24h 세션 복원 지원.

IN 메시지 + history + state + sessionId → OUT 자연어 응답 + 업데이트된 슬롯 state
CHATSTREAM MULTI-TURN RL · intake-long + /confirm
AI #03 · /api/ai/policy-analyze
Policy Reader
약관 분석기
admin / insurer

업로드된 보험 약관/증권 PDF에서 보장 항목, 면책 사유, 한도, 조건을 구조화해 추출합니다. PDF는 Claude의 document 블록으로 직접 전달, 별도 텍스트 추출 파이프라인 없음. 어드민 쿠키 또는 보험사 인증 필요.

IN PDF (multipart) + 선택 정책 코드 → OUT coverage[] / exclusions[] / limits[] JSON
PDFJSON RL · heavyAUTH
AI #04 · /api/ai/medical-analyze
Medical Analyze
의료기록 사전 분석
admin only

의료기록 PDF를 사전 분석해 진단·시술·합병증 후보를 추출합니다. PRD §6.2에 따라 어드민 전용, 환자별 해시만 감사 로그에 남기고 원본 PII는 기록하지 않습니다. DOWON_DISABLE_EXTERNAL_AI=true로 폐쇄망 모드.

IN 의료기록 PDF + 환자 식별자(해시) → OUT diagnoses / procedures / complications JSON
PDFPHI-SAFE RL · heavyAUTH ON-PREM OPT
AI #11 · /api/ai/coverage-check
Coverage Check
보험금 가능성 셀프체크
public

의뢰인 약관 + 사고 정보를 매칭해 보장·면책 후보 조항을 안내합니다. 변호사법 §23 / 보험업법 준수를 위해 단정 표현 금지. 최근 업데이트로 멀티턴 follow-up(텍스트/PDF 추가)을 지원해 처음부터 다시 시작하지 않고 검토를 갱신할 수 있습니다.

IN 약관 텍스트 또는 PDF + 사고 정보 + history? → OUT possibility · coverage · exclusions · 관련 판례
MULTI-TURN ✶ NEW PDFSTREAM RL · heavyAUDIT
B2B · /api/ai/subrogation-check
Subrogation Check
구상 가능성 진단
insurer

보험사 실무진이 사고 유형 · 당사자 · 손해액 · 지급 보험금 · 과실 비율 · 추가 사실을 입력하면 구상 청구 가능성과 검토 포인트를 진단합니다.

IN 사고 메타 + 손해/지급/과실 정보 → OUT 구상 가능성 점수 + 근거 + 추가 확인 자료 리스트
JSONRL · advice AUDIT
UTIL · /api/ai/lawyer-match
Lawyer Match
변호사 매칭
public

사건 카테고리 + 자유 서술을 받아 도원 변호사 프로필 DB에서 가장 적합한 후보를 선정합니다. Triage / Intake 종료 시점에 호출돼 추천 결과를 함께 제시하는 헬퍼 에이전트.

IN 카테고리 + 사건 요약 → OUT 변호사 후보 N명 + 매칭 사유
JSONRL · search AUDIT
UTIL · /api/ai/library-search
Library Search
판례·칼럼 시맨틱 검색
public

도원 라이브러리(판례 / 칼럼 / 강의)를 임베딩 기반 시맨틱 검색합니다. Coverage Check 등 다른 에이전트에서 결과 카드에 관련 항목을 자동 매칭해 보여주는 데에도 사용됩니다.

IN 검색 쿼리 + 카테고리 필터 → OUT 관련도 정렬된 항목 리스트 (제목·발췌·URL)
EMBEDDINGS RL · searchAUDIT
